# Market Basket Analysis using Apriori

## Overview
This project applies the Apriori algorithm on retail transaction data to identify frequent itemsets and generate association rules.

## Tech Stack
- Python
- Pandas
- Mlxtend

## Features
- Frequent itemset generation
- Association rule mining (support, confidence, lift)
- Insights for cross-selling strategies

## Results
- Identified strong product combinations
- Helped improve marketing and inventory decisions
